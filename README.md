## How to Get Started

1. **Enter the student's actual score** in the "Actual Score" field. Be accurate!

2. **Provide the class's highest score** (maximum score) in the "Maximum Score" field. This sets the benchmark.

3. **Specify the total class size** (class strength) in the "Total Strength" field. This is crucial for relative grading.

4. **Click the "Calculate Relative Grade" button** and witness your unique relative grade displayed in vivid colors. It's like painting with data.
